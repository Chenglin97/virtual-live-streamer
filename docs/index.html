Capabilities
What it does
🎭

3D Avatar + Lip Sync

Full-body animated character with real-time viseme-based lip sync driven by word-level TTS timestamps.

🧠

Hermes Agent Brain

Powered by Hermes Agent with 27+ tools, persistent memory, skills system, and any LLM backend.

💬

Twitch Chat Integration

Reads viewer messages via IRC, responds in chat AND speaks through the avatar simultaneously.

💾

Never Repeats

Persistent memory tracks every response. Recent lines injected as "DO NOT REPEAT" context.

🎭

Mood Expressions

Happy, sad, angry, neutral — avatar expressions and TTS voice tone adapt to conversation mood.

🗣️

Idle Conversation

When chat is quiet, the streamer shares thoughts, asks questions, tells stories autonomously.

📝

Custom Persona

Edit SOUL.md to define personality, interests, speaking style. Hot-reloaded every message.

💻

Runs on a Laptop

No GPU required. Three Python processes + a browser. Works on Mac, Linux, any machine.

Skill Ecosystem
BotLearn Integration
Packaged as a BotLearn skill. Any Hermes Agent can install it and become a streamer.

The streamer is a skill

The entire virtual streamer is packaged as @botlearn/virtual-live-streamer — a standard BotLearn skill that any Hermes Agent can install.

  • Install on any Hermes Agent instance with one command
  • Follows BotLearn format: SKILL.md, manifest.json, knowledge/, strategies/, tests/
  • Agents can learn streaming as a capability, not just a one-off script
  • Skills are shareable through BotLearn's social learning network
  • Other agents can install this skill and become streamers themselves
  • Self-improving: Hermes can refine its own streaming behavior over time

Skill structure

@botlearn/virtual-live-streamer/
  SKILL.md — role definition & boundaries
  manifest.json — metadata & deps
  knowledge/
    streaming-setup.md
  strategies/
    persona-management.md
    chat-response.md
  tests/
    smoke.json — 5 validation checks
    benchmark.json — 10-task suite

The skill encodes domain knowledge (how streaming works), strategies (how to respond to chat, manage persona), and tests (verification the skill works) — everything an agent needs to become a streamer.
